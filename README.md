# Cidadão Ativo

> Projeto front-end simples para coleta e acompanhamento de problemas de infraestrutura comunitária.

## Estrutura do projeto

- `Estilo/` — folhas de estilo CSS (style.css).
- `IMG/` — imagens usadas no site.
- `index/` — páginas HTML (ex.: `index.html`, `Formulário infra.html`).

## Como visualizar localmente

1. Abra o arquivo `index/index.html` no navegador (duplo-clique ou arraste para o navegador).
2. Para editar, abra o projeto no VS Code e altere os arquivos em `index/` e `Estilo/`.
